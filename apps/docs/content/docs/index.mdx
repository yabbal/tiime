Tiime est un ensemble d'outils TypeScript pour la comptabilite [Tiime](https://www.tiime.fr) :

- **`tiime-sdk`** — SDK TypeScript pour integrer Tiime dans vos applications
- **`tiime-cli`** — CLI pour piloter votre compta depuis le terminal

<Callout type="warn" title="Projet non officiel">
  Ce projet est personnel et experimental. Il n'est pas affilie a, ni approuve par Tiime.
</Callout>

## Choisissez votre outil

<Tabs items={["SDK TypeScript", "CLI"]}>
  <Tab value="SDK TypeScript">
    <Steps>
      <Step>
        ### Installer le SDK

        ```bash title="Terminal"
        npm install tiime-sdk
        ```
      </Step>

      <Step>
        ### Configurer l'authentification

        ```bash title="Terminal"
        export TIIME_EMAIL=vous@example.com
        export TIIME_PASSWORD=votre-mot-de-passe
        export TIIME_COMPANY_ID=12345
        ```
      </Step>

      <Step>
        ### Utiliser

        ```ts title="app.ts"
        import { TiimeClient } from "tiime-sdk";

        const client = new TiimeClient();

        const invoices = await client.invoices.list({ status: "paid" });
        const balances = await client.bankAccounts.balance();
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab value="CLI">
    <Steps>
      <Step>
        ### Installer le CLI

        <Tabs items={["npm", "Homebrew"]}>
          <Tab value="npm">
            ```bash title="Terminal"
            npm install -g tiime-cli
            ```
          </Tab>
          <Tab value="Homebrew">
            ```bash title="Terminal"
            brew tap yabbal/tap && brew install tiime
            ```
          </Tab>
        </Tabs>
      </Step>

      <Step>
        ### Se connecter

        ```bash title="Terminal"
        tiime auth login
        tiime company use --id 12345
        ```
      </Step>

      <Step>
        ### Utiliser

        ```bash title="Terminal"
        tiime status
        tiime invoices list --status paid --format table
        tiime bank balance
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>

## Sortie JSON native

Toutes les commandes CLI retournent du JSON par defaut, combinable avec `jq` :

```bash title="Terminal"
# Total des factures payees
tiime invoices list --status paid --all | jq '[.[].total_including_taxes] | add'

# Transactions > 1000 EUR
tiime bank transactions --all | jq '[.[] | select(.amount > 1000)]'

# Autres formats
tiime invoices list --format table   # Tableau ASCII
tiime invoices list --format csv     # Export CSV
```
