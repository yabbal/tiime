## SDK TypeScript

```bash title="Terminal"
npm install tiime-sdk
```

```ts title="app.ts"
import { TiimeClient } from "tiime-sdk";

const client = new TiimeClient({
  email: "vous@example.com",
  password: "votre-mot-de-passe",
  companyId: 12345,
});

const invoices = await client.invoices.list({ status: "paid" });
```

<Callout type="info">
  Le SDK fonctionne de maniere autonome. Voir [Authentification SDK](/docs/sdk/auth) pour tous les modes d'auth.
</Callout>

## CLI

### Installer

<Tabs items={["npm", "Homebrew", "Sources"]}>
  <Tab value="npm">
    ```bash title="Terminal"
    npm install -g tiime-cli
    ```
  </Tab>
  <Tab value="Homebrew">
    ```bash title="Terminal"
    brew tap yabbal/tap
    brew install tiime
    ```
  </Tab>
  <Tab value="Sources">
    ```bash title="Terminal"
    git clone https://github.com/yabbal/tiime.git
    cd tiime
    pnpm install && pnpm build
    pnpm link --global
    ```
  </Tab>
</Tabs>

Verifiez l'installation :

```bash title="Terminal"
tiime --help
```

### Configurer

<Steps>
  <Step>
    ### Authentification

    Mode interactif :

    ```bash title="Terminal"
    tiime auth login
    ```

    Mode script / CI :

    ```bash title="Terminal"
    tiime auth login --email vous@example.com --password votre-mot-de-passe
    ```
  </Step>

  <Step>
    ### Selection de l'entreprise

    ```bash title="Terminal"
    tiime company list
    tiime company use --id 12345
    ```

    Verifiez l'entreprise active :

    ```bash title="Terminal"
    tiime company get
    ```
  </Step>
</Steps>

### Autocompletion shell

<Tabs items={["zsh", "bash", "fish"]}>
  <Tab value="zsh">
    Ajoutez a `~/.zshrc` :
    ```bash title="~/.zshrc"
    eval "$(tiime completion --shell zsh)"
    ```
  </Tab>
  <Tab value="bash">
    Ajoutez a `~/.bashrc` :
    ```bash title="~/.bashrc"
    eval "$(tiime completion --shell bash)"
    ```
  </Tab>
  <Tab value="fish">
    ```bash title="Terminal"
    tiime completion --shell fish | source
    ```
  </Tab>
</Tabs>

### Langue

L'aide s'affiche dans la langue de votre systeme (francais ou anglais).

Pour forcer une langue :

```bash title="Terminal"
TIIME_LANG=fr tiime --help   # Francais
TIIME_LANG=en tiime --help   # English
```

<Callout type="info" title="Variable d'environnement">
  Exportez `TIIME_LANG` dans votre profil shell pour un changement permanent.
</Callout>
