Le CLI Tiime offre un ensemble complet de commandes pour gérer votre comptabilité.

## Résumé rapide

```bash
tiime status    # Résumé : soldes, factures en attente, transactions récentes
```

## Formats de sortie

Toutes les commandes de listing supportent `--format` :

| Format | Description | Usage |
|--------|-------------|-------|
| `json` | JSON structuré (défaut) | Scripts, jq, agents IA |
| `table` | Tableau ASCII | Lecture humaine |
| `csv` | Valeurs séparées par des virgules | Export tableur |

## Outils

```bash
tiime open                  # Ouvrir Tiime dans le navigateur
tiime open invoices         # Ouvrir la section factures
tiime version               # Afficher la version
```
