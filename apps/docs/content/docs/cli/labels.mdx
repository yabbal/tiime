## Labels personnalisés

```bash title="Terminal"
tiime labels list
```

## Labels standards

Labels du plan comptable français :

```bash title="Terminal"
tiime labels standard
```

<Callout type="info">
  Les labels standards correspondent aux catégories comptables officielles (loyer, restaurant, fournitures, etc.).
</Callout>

## Tags

```bash title="Terminal"
tiime labels tags
```
