---
title: Notes de frais
description: Lister, consulter et créer des notes de frais
---

## Lister

```bash title="Terminal"
tiime expenses list
tiime expenses list --format table
```

## Consulter

```bash title="Terminal"
tiime expenses get --id 5
```

## Créer

```bash title="Terminal"
tiime expenses create --name "Déplacement client"
```
